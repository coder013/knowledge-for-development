# MariaDB
- 오픈 소스 기반의 RDBMS
- 오라클이 MySQL을 인수한 후, 상업용 라이선스를 유료화하는 것에 반발한 개발자들이 오라클을 나온 후 MariaDB를 개발
- MySQL 5.5 버전을 포크
- MySQL과의 높은 호환성(100%에 가까운)과 최적화를 통해 더 뛰어난 성능을 가지고 있음
- 오라클로부터 자유로움
