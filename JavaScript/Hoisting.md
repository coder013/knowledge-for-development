# Hoisting
 - 함수 안의 변수 및 함수 선언을 해당 함수의 최상단으로 끌어올리는 것
 - 선언만 끌어올려지고 초기화는 끌어올려지지 않음
 - 변수 선언이 함수 선언보다 위로 끌어올려짐
 - var에서만 발생, let, const에서는 발생하지 않음
