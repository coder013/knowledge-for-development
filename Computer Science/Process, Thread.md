# Process, Thread

## 프로그램(Program)
- 특정 작업을 실행하기 위해 컴퓨터에 명령을 내리는 문장의 모음
- 쉽게 말하자면 특정 작업을 실행하기 위한 파일
- 실행되지 않은 정적인 상태

## 프로세스(Process)
- 실행 중인 프로그램
- 프로그램을 실행하면 해당 파일이 메모리에 할당됨. 이 시점부터 프로세스라고 부름
- 동적인 상태
