# 렌더링 엔진
## Webkit
 - HTML 문서 파싱 => DOM(문서 객체 모델) 트리 생성
 - HTML 문서를 읽던 중 Style Sheets(CSS)를 만날 경우 파싱 => CSSOM(CSS 객체 모델) 트리 생성
 - DOM 트리와 CSSOM 트리를 합쳐 렌더 트리 생성
 - 렌터 트리 화면에 배치
 - 렌터 트리 화면에 그리기
