# HTTP

## HTTP란?
- HTTP(HyperText Transfer Protocol, 하이퍼텍스트 전송 프로토콜)
- 웹 서버와 클라이언트간의 통신을 위한 프로토콜
- TCP/IP를 기반으로 한 응용 계층 프로토콜
- 연결 상태를 유지하지 않는 비연결성 프로토콜

## 동작 방식
- 클라이언트가 요청(Request)을 보내면 서버는 해당 요청에 맞는 응답(Response)을 보냄
